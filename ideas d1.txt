# Big-picture theme

**“Clinical systems, human-first”** — Apple-style minimalism + clinical trust.

* Visual language: lots of whitespace, rounded cards, frosted-glass panels, soft shadows, and calm gradients (no harsh colors).
* Tone: concise, confident, patient-centered.
* Motif: subtle hospital / lab iconography (stethoscope, clipboard, microchip) used sparingly as line icons.
* Colors: soft neutrals + a single accent (teal / ocean blue) that reads clinical and modern.

# How to present each project (case study formula)

Always follow: **Context → Problem → Your Role → Solution (tech + clinical) → Outcome (metrics) → Artifacts**
Example structure for the Health Camp site:

* Title: Digital Health Camp — World Pharmacist Day 2025
* Context: College camp screening 400 patients in 1 day.
* Problem: Manual bookings, lost forms, slow report generation.
* Role: Full-stack — UI, Firestore, admin dashboard, PDF report generator.
* Solution: Appointment system + station-specific forms + PDF reports (show snippets of code or architecture diagram).
* Outcome: Reduced check-in time 50%, 400 registrations, instant reports to patients.
* Download: “View live demo” or “Download project summary (PDF)”.

# Image & asset guidance

* Use real screenshots: homepage, admin dashboard, patient-report, station page. Crop them into rounded rectangles and place inside mock devices.

# Typography & spacing

* Use system stack: `font-family: system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial;`
* Big headline (clamp), medium subheads, long-form body ~16px with 1.5 line height.
* Generous vertical spacing (48–96px between sections).
